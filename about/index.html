I love it mostly to work with the following:

  • git and Github
  • Linux (Ubuntu, Debian) and the command line
  • web development (HTML5, Javascript, Typescript, CSS3, Vaadin)
  • client/server/network programming
  • 2D and 3D graphics and math
  • libGDX, Slick2D, Unity
